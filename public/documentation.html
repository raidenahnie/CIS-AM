Introduction

CIS-AM (Curriculum Implementation System - Attendance Monitoring) is a specialized web-based attendance management system developed for the Department of Education - Cavite Province, specifically designed for use by the Curriculum Implementation Division.

This system provides accurate, GPS-verified attendance tracking for DepEd Cavite personnel, ensuring accountability and efficient workforce management across multiple school locations and division offices throughout Cavite Province.

Configuration

Database Configuration

SQLite (Default)

DB_CONNECTION=sqlite # Database file created automatically at database/database.sqlite

MySQL

DB_CONNECTION=mysql DB_HOST=127.0.0.1 DB_PORT=3306 DB_DATABASE=cis_am DB_USERNAME=root DB_PASSWORD=your_password

Employee Usage Guide

User Account Registration

To register an account, please contact your system administrator to register you in the system.

1. Login

  1. Navigate to the login page
  2. Enter your email and password
  3. Click "Sign In"

2. Check In

  1. Select your workplace from the dashboard
  2. Click "Check In" button
  3. Allow location access when prompted
  4. System verifies you're within the workplace radius
Manual Entry Option

If GPS is unavailable, you can use the Manual Entry option with the access code provided by your administrator.

3. Break Management

  1. Click "Start Break" when taking a break
  2. Click "End Break" when returning to work
  3. Break duration is automatically calculated

4. Check Out

  1. Click "Check Out" when ending your work day
  2. System calculates total hours worked (minus break time)

5. View Attendance History

  1. Navigate to "Attendance History" section
  2. View detailed logs of all activities
  3. Filter by date range

Administrator Guide

1. Dashboard Overview

  • View real-time statistics (check-ins, late arrivals, etc.)
  • Monitor current employee status
  • Quick access to all management functions

2. User Management

  • Add new employees with role assignment
  • Edit user information and permissions
  • Deactivate/activate user accounts
  • Perform bulk operations for efficiency

3. Workplace Management

  • Create new workplace locations
  • Set GPS coordinates and check-in radius
  • Assign users to workplaces
  • View workplace statistics

4. Reports Generation

  1. Navigate to Reports section
  2. Select report type (Weekly/Monthly/Custom)
  3. Apply filters (employee, workplace, date range)
  4. Click "Generate Report"
  5. Export to CSV/Excel format

5. System Settings

  • Update manual entry access code
  • Configure admin account settings
  • Manage system parameters
Security Notice

All sensitive operations require admin password verification for security purposes.

API Reference

Authentication

Method Endpoint Description
POST /login User login
POST /logout User logout
POST /password/reset Password reset

Employee Dashboard APIs

Method Endpoint Description
GET /api/user-stats/{userId} User statistics
GET /api/attendance-history/{userId} Attendance history
GET /api/attendance-logs/{userId} Activity logs
GET /api/current-status/{userId} Current work status
POST /api/checkin Check in
POST /api/perform-action Check out/break actions
GET /api/manual-entry-code Get manual entry access code

Admin Dashboard APIs

Method Endpoint Description
GET /admin/dashboard Admin dashboard
GET /admin/users User list
POST /admin/users Create user
PUT /admin/users/{id} Update user
DELETE /admin/users/{id} Delete user
GET /admin/workplaces Workplace list
GET /admin/attendance-stats Attendance statistics

Reports APIs

Method Endpoint Description
GET /admin/reports/attendance Generate attendance reports
GET /admin/reports/individual/{user} Individual employee report
GET /admin/reports/export Export report (CSV/Excel)
GET /admin/reports/summary-stats Summary statistics
